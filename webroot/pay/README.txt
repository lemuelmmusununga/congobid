=== TWPG payment gateway for PHP ===

== Description ==

This plugin extends PHP applications allowing user to take payments directly on your store via TWPG API.


<h4>Others PHP Payment module</h4>

== Description ==

1) put all in same directory
2) plugin_Ecom.php is the main plugin file
3) others are for testing orders example test_createOrder.php where you need to update your application data

Inside SSL folder set certificates sent by the bank.(ca.pem, cert.pem, key.pem)
